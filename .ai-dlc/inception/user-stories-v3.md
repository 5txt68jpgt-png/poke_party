# User Stories - v3: バトル展開ガイド

## US-01: バトルガイド生成（シングル）
**As a** ポケモン初心者の親
**I want** シングルバトル用の戦い方ガイドを見たい
**So that** 子供とのごっこ遊びで盛り上がれる

### Acceptance Criteria
- [ ] パーティ生成時にバトルガイドが同時に生成される
- [ ] ガイドにはポケモンの役割（アタッカー、サポート等）と出し順が含まれる
- [ ] 技の詳細には踏み込まず、ポケモン中心の説明
- [ ] 4〜5文程度の簡潔な内容
- [ ] 日本語で生成される

---

## US-02: バトルガイド生成（ダブル）
**As a** ダブルバトルで遊びたい親
**I want** ダブルバトル用の戦い方ガイドを見たい
**So that** 2vs2の連携を理解して遊べる

### Acceptance Criteria
- [ ] ダブルモードで生成時、ダブル向けガイドが生成される
- [ ] 先発2匹の組み合わせが提案される
- [ ] 連携技や味方との相性についてアドバイスがある
- [ ] 4〜5文程度の簡潔な内容

---

## US-03: ガイド表示/非表示
**As a** ユーザー
**I want** 戦い方ガイドを表示/非表示できる
**So that** 必要なときだけ確認できる

### Acceptance Criteria
- [ ] 「戦い方を見る」ボタンが表示される
- [ ] ボタンをタップするとガイドが展開される
- [ ] もう一度タップすると閉じる
- [ ] アニメーションでスムーズに開閉する

---

## US-04: ガイド再生成
**As a** ユーザー
**I want** パーティ再生成時にガイドも更新される
**So that** 常に現在のパーティに合ったガイドが見られる

### Acceptance Criteria
- [ ] パーティ再生成時、ガイドも新しく生成される
- [ ] 古いガイドは破棄される
- [ ] 再生成後もガイドの表示/非表示状態は保持される
