# User Stories - PokeParty

## Epic: パーティ生成

### US-1: テーマを指定してパーティを生成する
**As a** ポケモン知識のない親
**I want to** テーマを自然言語で入力してパーティを生成したい
**So that** 子供とのごっこ遊びに使うポケモンをすぐに決められる

**受け入れ基準：**
- [ ] テキスト入力欄にテーマを入力できる
- [ ] 「第一世代のみ」「空を飛ぶポケモン」「リザードンを入れて」などの自然言語を受け付ける
- [ ] 生成ボタンを押すとパーティが生成される
- [ ] 生成中はローディング表示がされる
- [ ] エラー時は適切なメッセージが表示される

---

### US-2: パーティの匹数を指定する
**As a** 親
**I want to** パーティのポケモン数を指定したい
**So that** 1vs1や3vs3など、遊び方に合わせたパーティを作れる

**受け入れ基準：**
- [ ] 1〜6匹の範囲で匹数を選択できる
- [ ] デフォルトは3匹
- [ ] 指定した匹数のパーティが生成される

---

### US-3: 生成されたパーティを確認する
**As a** 親
**I want to** 生成されたポケモンと技の一覧を見たい
**So that** ごっこ遊びで何を出せばいいかわかる

**受け入れ基準：**
- [ ] 各ポケモンの日本語名が表示される
- [ ] 各ポケモンのタイプが表示される
- [ ] 各ポケモンの技4つが日本語名で表示される
- [ ] 各技のタイプが表示される
- [ ] ポケモンの画像（スプライト）が表示される

---

### US-4: パーティを再生成する
**As a** 親
**I want to** 気に入らなければパーティを再生成したい
**So that** より納得のいくパーティで遊べる

**受け入れ基準：**
- [ ] 再生成ボタンがある
- [ ] 同じテーマ・匹数で別のパーティが生成される

---

## Epic: バトル補助

### US-5: タイプ相性を確認する
**As a** 親
**I want to** 技とポケモンの相性を確認したい
**So that** 「こうかはばつぐんだ！」を正しく言える

**受け入れ基準：**
- [ ] 攻撃側の技タイプを選択できる
- [ ] 防御側のポケモンタイプを選択/入力できる
- [ ] 相性結果が表示される（ばつぐん/ふつう/いまひとつ/こうかなし）
- [ ] 倍率も表示される（2倍、0.5倍など）

---

### US-6: 技の詳細を確認する
**As a** 親
**I want to** 技の詳細情報を見たい
**So that** 技を出す真似をするときの参考にできる

**受け入れ基準：**
- [ ] 技をタップすると詳細が表示される
- [ ] 技のタイプが表示される
- [ ] 技の分類（物理/特殊/変化）が表示される
- [ ] 技の簡単な説明が表示される（日本語）

---

## Epic: UI/UX

### US-7: スマホで快適に操作する
**As a** 親
**I want to** スマホで片手で操作したい
**So that** 遊びながらでも使える

**受け入れ基準：**
- [ ] スマホ画面に最適化されたレイアウト
- [ ] タップしやすいボタンサイズ（44px以上）
- [ ] スクロールで全ての情報にアクセスできる
- [ ] 横向き・縦向き両対応

---

### US-8: 遊び中にすぐ情報を見つける
**As a** 親
**I want to** 必要な情報にすぐアクセスしたい
**So that** ごっこ遊びのテンポを崩さない

**受け入れ基準：**
- [ ] パーティ一覧が見やすく整理されている
- [ ] 現在のポケモンを1タップで切り替えられる
- [ ] 技リストが常に見える状態

---

```
     |                    |
  US-3 パーティ確認    US-8 情報アクセス
     |
  US-4 再生成
     |

```

## 優先度

| 優先度 | ストーリー | 理由 |
|--------|-----------|------|
| Must | US-1, US-3 | コア機能、これがないと始まらない |
| Must | US-2 | 1vs1〜6vs6の柔軟な対応は必須 |
| Must | US-7 | スマホで使えないと価値がない |
| Should | US-4 | 使い勝手向上 |
| Should | US-5 | ごっこ遊びの質向上 |
| Could | US-6, US-8 | あると便利 |
