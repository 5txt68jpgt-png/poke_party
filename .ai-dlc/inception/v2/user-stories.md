# User Stories v2 - パーティ生成の改善

## ストーリー一覧

### US-V2-1: おまかせ生成（Must）

**ストーリー**
親として、テーマを考えずにボタン1つでパーティを生成したい。そうすれば、寝る前の限られた時間でもすぐにごっこ遊びを始められる。

**受け入れ基準**
- [ ] 「おまかせ生成」ボタンがある
- [ ] ボタンを押すとテーマとパーティが同時に生成される
- [ ] 生成されたテーマが画面に表示される
- [ ] 1回のAPIコールで完結する

---

### US-V2-2: 意外性のあるパーティ（Must）

**ストーリー**
親として、毎回違う意外なポケモンに出会いたい。そうすれば、子供と「こんなポケモンいるんだ！」と盛り上がれる。

**受け入れ基準**
- [ ] 同じテーマで複数回生成しても、毎回違うポケモンが出る
- [ ] 御三家や代表的なポケモンばかりにならない
- [ ] テーマに合わないポケモンは出ない

---

### US-V2-3: テーマ入力との両立（Must）

**ストーリー**
親として、特定のテーマでパーティを作りたい時もある。おまかせとテーマ入力を使い分けたい。

**受け入れ基準**
- [ ] 既存のテーマ入力機能が使える
- [ ] おまかせ生成とテーマ入力の両方がUIに存在する
- [ ] どちらを使うか直感的にわかる

---

### US-V2-4: 再生成での変化（Should）

**ストーリー**
親として、再生成ボタンを押したら違うパーティが出てほしい。同じポケモンばかりだとつまらない。

**受け入れ基準**
- [ ] 再生成で異なるポケモンが選ばれる確率が高い
- [ ] テーマが同じでも結果が変わる

---
