# User Stories v3 - ポケモン検索機能

## ストーリー一覧

### US-V3-1: ポケモン名で検索（Must）

**ストーリー**
親として、子どもが選んだポケモンの名前を入力して検索したい。そうすれば、知らないポケモンでもタイプがわかる。

**受け入れ基準**
- [ ] 相性チェックモーダル内にポケモン検索欄がある
- [ ] 日本語でポケモン名を入力できる
- [ ] 入力中にリアルタイムで候補が表示される
- [ ] 全世代のポケモンが検索対象

---

### US-V3-2: 検索結果からタイプ自動選択（Must）

**ストーリー**
親として、検索候補からポケモンを選んだら、そのタイプが自動で選択されてほしい。手動でタイプを選ぶ手間を省きたい。

**受け入れ基準**
- [ ] 検索候補をタップするとそのポケモンのタイプが自動選択される
- [ ] 複合タイプ（2つのタイプ）にも対応
- [ ] タイプ選択後、相性結果が即座に表示される

---

### US-V3-3: タイプ直接選択との併用（Must）

**ストーリー**
親として、ポケモン検索とタイプ直接選択を使い分けたい。状況に応じて便利な方を選びたい。

**受け入れ基準**
- [ ] 既存のタイプ選択UIが残っている
- [ ] ポケモン検索とタイプ選択が同じモーダル内にある
- [ ] どちらを使っても相性チェックができる

---

### US-V3-4: 検索候補の視認性（Should）

**ストーリー**
親として、検索候補が見やすく表示されてほしい。ポケモンのタイプも候補に表示されるとより便利。

**受け入れ基準**
- [ ] 検索候補にポケモン名が表示される
- [ ] 検索候補にタイプバッジも表示される
- [ ] 候補は適切な件数（5〜8件程度）に絞られる

---
