# User Stories v4 - おまかせ生成改善 & ゲーム風デザイン

## ストーリー一覧

### US-V4-1: おまかせ生成でポケモン数を選択（Must）

**ストーリー**
親として、おまかせ生成でもポケモンの数を選びたい。子どもとのバトルに合わせて柔軟に対応したい。

**受け入れ基準**
- [ ] おまかせ生成時に1〜6匹を選択できる
- [ ] 選択UIはテーマ生成と同じ操作感
- [ ] デフォルトは3匹（現状維持）

---

### US-V4-2: ポケモンゲーム風の見た目（Must）

**ストーリー**
親として、アプリがポケモンゲーム風の見た目だと嬉しい。子どもと一緒に使う時、より楽しい雰囲気になる。

**受け入れ基準**
- [ ] 全体的にポケモンゲーム（最新作）風のデザイン
- [ ] カラースキームがゲーム風
- [ ] モダンで洗練された印象

---

### US-V4-3: 統一感のあるUI（Should）

**ストーリー**
親として、アプリ全体で統一感のあるデザインが良い。ページごとに雰囲気が違うと違和感がある。

**受け入れ基準**
- [ ] 全ページで統一されたデザイン言語
- [ ] ボタン、カード、モーダルのスタイルが統一
- [ ] フォント、色使いが一貫

---

### US-V4-4: モバイルで使いやすい（Must）

**ストーリー**
親として、スマートフォンで快適に使いたい。寝る前の布団の上でも片手で操作したい。

**受け入れ基準**
- [ ] モバイルファーストのレイアウト
- [ ] タッチしやすいボタンサイズ（44px以上）
- [ ] スクロールが自然
