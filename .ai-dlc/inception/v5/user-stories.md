# User Stories v5 - バトルテンポ改善

## ストーリー一覧

### US-V5-1: 相手ポケモンの選択保持（Must）

**ストーリー**
親として、一度選んだ相手ポケモンを覚えていてほしい。技を変えるたびに検索し直すのは面倒。

**受け入れ基準**
- [ ] 相手ポケモンを選択すると、その選択が保持される
- [ ] 別の技の相性チェックを開いた時も、前回の相手ポケモンが表示される
- [ ] 必要に応じて相手ポケモンを変更できる
- [ ] パーティを再生成すると相手選択はリセットされる

---

### US-V5-2: 受けた技の相性チェック（Must）

**ストーリー**
親として、子どもの技が自分のポケモンに効果抜群かどうかを知りたい。リアクションのためにすぐ確認したい。

**受け入れ基準**
- [ ] 自分のポケモンカードに「受けた技をチェック」ボタンがある
- [ ] ボタンを押すと技検索モーダルが開く
- [ ] 技名を入力すると候補がリアルタイム表示される
- [ ] 技を選択すると、自分のポケモンへの相性が表示される

---

### US-V5-3: 技名で検索（Must）

**ストーリー**
親として、技名を入力するだけでタイプを調べたい。技のタイプは覚えていないことが多い。

**受け入れ基準**
- [ ] 日本語で技名を入力できる
- [ ] 部分一致で候補が表示される
- [ ] 候補には技名とタイプが表示される
- [ ] 全世代の技が検索対象

---

### US-V5-4: 相性結果の視認性（Should）

**ストーリー**
親として、相性結果がすぐにわかるようにしてほしい。「こうかはばつぐんだ！」と言いたい。

**受け入れ基準**
- [ ] 効果抜群/今ひとつ/効果なしが明確に表示される
- [ ] 倍率も表示される
- [ ] 見た目が既存の相性表示と統一されている
