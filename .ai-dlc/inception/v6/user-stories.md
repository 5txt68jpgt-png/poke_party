# User Stories v6 - 技の入れ替え機能

## ストーリー一覧

### US-V6-1: 技の入れ替え（Must）

**ストーリー**
親として、AIが選んだ技を別の技に入れ替えたい。子どもが使いたい技があるときにすぐ対応したい。

**受け入れ基準**
- [ ] 技リストの技をタップすると入れ替えモーダルが開く
- [ ] そのポケモンが覚えられる技の候補が表示される
- [ ] 技を選択すると、元の技と入れ替わる
- [ ] 入れ替え後も相性チェック機能は使える

---

### US-V6-2: 技の詳細表示（Must）

**ストーリー**
親として、技を選ぶときに威力や分類を見たい。どの技が強いかわからないから参考にしたい。

**受け入れ基準**
- [ ] 候補リストに技名が表示される
- [ ] 候補リストにタイプが表示される
- [ ] 候補リストに威力が表示される
- [ ] 候補リストに分類（物理/特殊/変化）が表示される

---

### US-V6-3: タイプフィルタ（Should）

**ストーリー**
親として、特定のタイプの技だけを見たい。技が多すぎると探すのが大変。

**受け入れ基準**
- [ ] タイプ別にフィルタリングできる
- [ ] フィルタを解除すると全技が表示される
- [ ] 選択中のフィルタがわかりやすく表示される

---

### US-V6-4: 既存技との連携（Must）

**ストーリー**
親として、入れ替えた技でも相性チェックしたい。機能が分断されると使いにくい。

**受け入れ基準**
- [ ] 入れ替えた技も技リストに正常に表示される
- [ ] 入れ替えた技をタップすると相性チェックモーダルが開く
- [ ] 相性チェック結果が正しく表示される

---
